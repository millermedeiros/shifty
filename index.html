Shifty!

Shifty is a tweening engine built in JavaScript.  It is designed to fit any number of tweening needs. Features include:

  • Excellent performance; all design decisions were built around minimizing performance overhead.
  • Works well as the base Function in a prototype chain.
  • Modular, extensible architecture allows you to use only the Shifty components you need.
  • Small download: Full library (with all extensions) is about 3K minified and gzipped.  The core of the library (shifty.core.js) is just under 1.5K.

Shifty is an excellent animation solution for scenarios in which jQuery's animate() function is either unavailable or insufficient. It's particularly effective for creating <canvas> animations. A common use for Shifty is generating the steps of a Tween, as it is used to do in Joe Lambert's Morf library. Using Shifty in this way would also be very useful in a non-browser environment such as Node.js.

Shifty exposes only one Function globally: Tweenable(). To create a simple tween, you could have something like this:

    var myTweenable = new Tweenable();

    myTweenable.tween({

      from: {
  'x': 0
      },

      to: {
  'x': 10
      },

      'duration': 1000,

      'callback': function () {
  console.log('Hooray, all done!  Final value: ' + this.x);
      }

    });

This snippet would start a tween on the x variable from 0 to 1 over one second, and then log a happy message. You could have as many properties in a tween as you'd like.

Shifty can do quite a bit more than this. It gives you play control of a tween (stopping, pausing, and resuming), property access, as well as hooks and filters to customize each tween to your specific needs. In addition to this, there are several extensions that have been built for Shifty. These extensions primarily make Shifty more useful on the DOM by providing support such as CSS units and colors. All of this functionality is documented and demonstrated in the project repo on Github.

A collection of Robert Penner's easing formulas have been adapted for use with Shifty and are included in the standard download.
